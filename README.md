## Array Helper Methods
1. <b>`forEach`</b>
  * Accepts an anonmyous function as a parameter and executes code against each index in the array. This replaces the standard `for` loop     when iterating over an entire array.
2. <b>`map`</b>
3. <b>`filter`</b>
4. <b>`find`</b>
5. <b>`every`</b>
6. <b>`some`</b>
7. <b>`reduce`</b>
